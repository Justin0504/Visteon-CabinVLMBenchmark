# 🎯 车外（Exterior）3000 做到极致 — 大计划

> 目标：把 Category 2 车外 8 类做到**最高质量 + 测试集最佳表现 + 无退化 + 可演示**。暂不管 in-cabin。

## 一、现状（诚实）
- **数据**：8,030 / 5 源（exterior 2258、cars 1997、signs 1497、ocr 1500、landscape 778）——量已超 3000，**重点转质量**。
- **模型**：v1 最佳（cars 92.5 / signs 100 / exterior 8.0）；**v2 加 OCR+景观后退化**（cars→52.5、ocr→70）= 数据混合干扰。
- **覆盖**：7/8 类有数据，POI = 仅设计。
- **已知问题**：①无脑混合会退化；②各版本测试集不一致、不可比；③Qwen2-VL 标注器在车外偏弱（"Unknown"）；④自评裁判有偏置。

## 二、五个阶段

### E1 — 建立评测严谨性（地基，最优先）
- **冻结一个权威测试集**：每类 40 张、5 类共 200，**只抽一次、永不进训练**，所有版本都在它上面评 → 苹果对苹果。
- **客观指标**：车型/标志/景观用**真值准确率**；车外/OCR 用更强裁判（降自评偏置）。
- 验收：`heldout_frozen.json` + 评测脚本，base/v1/v2/v3 同集对比。

### E2 — 数据质量拉满
- **更强标注器重标车外 nuScenes**（若实验证明 Qwen2.5-VL 更好）→ 降 "Unknown"、答案更丰富。
- **均衡混合**：抑制主导源、上采样弱类；OCR 短答风格单独处理，避免带偏车型。
- **补覆盖**：建筑/地标加专门数据；OCR 考虑驾驶域文字。
- **人工抽检**每类一小批（IAA 式质检）。
- 验收：均衡高质 3000+ 车外集，各类 "Unknown" <5%。

### E3 — 模型优化（修退化 + 冲最佳）
- **均衡混合训练 v3**：加权采样，任何类不主导 → 修 cars/ocr 退化。
- **超参扫**：LoRA rank(8/16/32)、epoch(2/3)、LR → 冻结集上选最优。
- 每改一项 → 冻结集复评 → 留最好（论文消融表）。
- 验收：v3 在冻结集上**所有类 ≥ v1（无退化）**，OCR 恢复、车型/标志/景观高。

### E4 — POI MVP（最后一个能力）
- 实现 RAG MVP-A：图 → OCR 店招 → 高德/Google POI API → 整合回答。
- 验收：现场演示"看店招→查 POI→回答"闭环。

### E5 — 终展打磨
- Demo（已就绪，换最优模型）+ 海报（车外故事 + 分类别战果）+ 视频 + 论文车外章节。

## 三、时间线（到 8/5 Expo）
- **本周**：E1 冻结测试集 → E2 标注器升级+均衡 → E3 训 v3 修退化；
- **下周**：E4 POI MVP + 持续调到最佳；
- **7/13 DR#3** 前：E5 打磨 + 演示视频。
