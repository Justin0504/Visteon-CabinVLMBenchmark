# 📈 第二轮优化总结 · 车外数据线均衡化 + 微调提升（袁傲杰）

> 更新 2026-06-11 · 在第一轮（2258 车外图文对 + bootstrap v0）基础上，本轮做"查漏补缺 + 效果优化"。

## 一、本轮核心成果

**1. 数据：从单源偏科 → 5 源均衡，8,030 图文对**（达成 7 月底 8000 目标·数量）
| 源 | 数量 | 补的能力 |
|---|---|---|
| nuScenes 车外 | 2,258 | 场景/行人/路况 |
| Stanford Cars | 1,997 | **车型识别** |
| GTSRB | 1,497 | **交通标志** |
| TextVQA | 1,500 | **文字/广告 OCR** |
| SUN397 景观 | 778 | **自然景观** |

**2. 关键方法：标签引导 recaption**——把数据集**真值标签灌进 prompt 当 ground truth**，模型生成**正确答案**（根治第一轮"Unknown"幻觉）。

**3. 微调效果（v1，per-category 客观评测）**：
| 类别 | base | bootstrap v1 | 提升 |
|---|---|---|---|
| **车型 make/model**（准确率）| 7.5% | **90.0%** | **+82.5** 🚀 |
| **交通标志**（准确率）| 95% | **100%** | +5 |
| 车外场景（judge 1–10）| 7.6 | 7.97 | 无退化 ✓ |

→ **证明标签引导微调能把最弱类从 7.5% 拉到 90%，且不损害已有能力（无灾难性遗忘）**。这是论文最硬的消融证据。
